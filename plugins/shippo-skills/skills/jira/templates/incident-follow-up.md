# Incident Follow-up Template

Use for post-incident remediation work.

```markdown
**Description:**
Follow-up from [INC-xxxx](link). [Brief description of the fix needed].

**Context:**
* Root cause: [1 sentence]
* Impact: [1 sentence]

**AC:**
* [Fix or mitigation deliverable]
```

## Field Defaults

| Field | Value |
|-------|-------|
| Issue Type | Task |
| Taxonomy | Incident Follow-up |
| Labels | `incident-follow-up`, `ready-for-grooming` |

## Rules

1. Always set taxonomy to "Incident Follow-up"
2. Always add `incident-follow-up` label
3. Reference INC-xxxx IDs in description
4. Link to parent epic/story if applicable
