# Story Template

Use for user-facing value and feature delivery.

```markdown
**Description:**
[User story from user perspective]. This [action] will [value delivered].

**Background:** [Why this matters].

**Scope:** [What's included]. [Delivery format].

**Story Content:**
* [Element 1]
* [Element 2]

**AC:**
* [User-focused criteria 1]
* [User-focused criteria 2]
```

## Field Defaults

| Field | Value |
|-------|-------|
| Issue Type | Story |
| Parent | Epic |
| Taxonomy | Feature - Product Enhancement |
| Labels | `ready-for-grooming` |
